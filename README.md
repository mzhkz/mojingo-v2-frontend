## WordLink-FrontEnd

### Using framework and libraries.
- Vue(2.6.11)
- Vue Router (3.1.5)
- Vuex (3.1.)
- jQuery (3.3.1)
- Axios (0.18.0)
- Webpack4 (4.20.2)

### Command
|Command|Description|
|:---|:---|
|yarn start|開発用サーバを起動する(ライブリロードあり)|
|yarn server|Expressを起動(事前に本番ファイルを生成する必要あり)|
|yarn build:dev|開発用のファイルを出力する|
|yarn build:prod|本番用のファイルを出力する|
